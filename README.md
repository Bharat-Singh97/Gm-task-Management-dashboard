# GM Task Management Dashboard

A role-based task management web application developed for North Western Railway (NWR).

## 🔧 Tech Stack
- JavaScript
- Firebase (Authentication + Firestore)
- HTML, CSS

## 🚀 Features
- Role-based login (GM, PHOD, HOD)
- Task assignment and forwarding
- Status tracking (Pending, Ongoing, Completed)
- Comments and communication
- Real-time updates using Firestore

## 📌 Description
This system helps railway management digitally assign, monitor, and track tasks across departments with improved transparency and efficiency.

## 🔐 Note
Firebase API keys are removed for security.
